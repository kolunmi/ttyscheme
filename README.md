# ttyscheme

ttyscheme features 150 popular terminal color schemes tuned specifically for use in the tty.

## Installation
```bash
git clone https://github.com/kolunmi/ttyscheme
cd ttyscheme
make install
```

## Examples
```bash
# list all schemes
ttyscheme -l

# apply a scheme
ttyscheme nord

# show all tty color combinations
ttyscheme -a
```
